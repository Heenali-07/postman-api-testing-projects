# FakeStore API Testing

## Overview
This project contains API testing scenarios for the FakeStore API using Postman.

## Features Covered
- GET Products
- GET Single Product
- POST Product
- PUT Product
- DELETE Product
- JSON Response Validation
- Status Code Validation
- Response Time Validation
- Assertions

## Tools Used
- Postman
- REST API
- JSON

- FakeStore API Validation.postman_collection.json
